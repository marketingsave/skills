# Examples

## Example 1 — B2B SaaS (project management)

- Domain DR 28, target page `/templates/sprint-planning`, target keyword "sprint planning template"
- Link Intersect across monday/asana/clickup → 180 domains → 34 A/B prospects
- Split: 12 guest-post, 8 resource-page, 6 broken-link, 5 digital-pr (survey), 3 skyscraper
- 6-week outcome: 7 placed links, avg DR 44, ranking 18 → 9, +180% organic to page

## Example 2 — Local service (HVAC, metro area)

- Domain DR 15, target page `/services/ac-repair-austin`
- Strategy: local citations + resource pages (chamber of commerce, neighborhood blogs, local news) + 1 data study ("Austin summer AC failure rates")
- 12-week outcome: 11 placed links, avg DR 38, map-pack entry for primary keyword
