# Tech Checklist — Event Mode (in-person / hybrid)

Load this when `mode = event`. Adapt to format (pure in-person vs. hybrid).

## Venue AV

```
[ ] Microphones tested — lapel for speakers, handheld for Q&A, backup of each
[ ] Mic channels assigned (document channel per speaker)
[ ] Sound system levels checked in empty room AND with audience simulation
[ ] Projector / screens tested with each presenter's laptop + dongle
[ ] Presentation clicker batteries fresh; backup clicker available
[ ] Stage lighting adequate (especially if recording)
[ ] Livestream encoder tested end-to-end (hybrid only)
[ ] Video recording source confirmed; backup recorder running
```

## Registration Desk

```
[ ] Printed attendee list (alphabetized) + backup digital copy on tablet
[ ] Name tags pre-printed for registrants + blank stock for walk-ins
[ ] Check-in app / QR scanner tested on venue wifi
[ ] Cash box / payment reader for day-of registrations (if applicable)
[ ] Welcome packet or program handout stack
[ ] Signage: arrows from entrance → registration → main room
```

## Signage + Wayfinding

```
[ ] Venue exterior sign (if unclear from street)
[ ] Registration directional signage
[ ] Room labels for breakouts
[ ] Sponsor signage placed per agreement
[ ] Restroom + exit signs visible
[ ] Agenda board near registration
```

## Catering / Hospitality

```
[ ] Final headcount confirmed with vendor 48 h out
[ ] Dietary restrictions list to vendor
[ ] Serving tables + linens + utensils confirmed
[ ] Water stations set up at entry + room perimeters
[ ] Service timing aligned with agenda (snacks during arrivals, meal at the break)
```

## In-Person Backup Plan

```
[ ] Backup mic + batteries accessible at stage
[ ] Printed slide handouts if projector fails
[ ] Mobile hotspot for payment terminal / check-in app
[ ] Venue contact phone + after-hours number
[ ] First-aid kit location known to all staff
[ ] Emergency exits briefed with volunteers
[ ] Weather contingency if outdoor (tent, indoor pivot)
```

## Hybrid-Specific

```
[ ] Virtual attendee waiting-room greeter assigned
[ ] Chat moderator pushing virtual questions to host
[ ] Remote speaker tech check 24 h out
[ ] In-room camera framing tested from virtual attendee perspective
[ ] Audio feed to stream (not just room mic) confirmed
```
