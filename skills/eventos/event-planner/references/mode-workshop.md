# Mode: Workshop (hands-on transformation)

## Core constraint

Every workshop has **one transformation**: State A → State B. Anything that does not move attendees from A to B is cut.

## Transformation-driven agenda (percentages of total time)

| Block | % | Purpose |
|---|---|---|
| Opening Hook | 5–10% | Grab attention, create safety, show the promise |
| Foundation | 15–20% | Teach the core framework |
| Exercise 1 | 15–20% | Apply framework to their own situation |
| Deep Dive | 15–20% | Advanced nuance or second concept |
| Exercise 2 | 15–20% | Realistic scenario combining everything |
| Q&A / Hot Seats | 10–15% | Address individual situations |
| Close + Next Steps | 5–10% | Summarize, assign action, soft pitch if any |

## Timing rules

- No lecture block > 15 min without interaction
- Every exercise has prompt + time limit + debrief method
- 5-min buffer for 90-min workshop; 10-min buffer for 3h+
- Virtual workshops insert a 5-min break every 45 min

## Exercise specification (required for every exercise)

- **Type:** solo / pair / small group / full room / hands-on
- **Exact prompt** (written verbatim, not summarized)
- **Duration + debrief time**
- **Materials** (worksheet page, tool, template)
- **Debrief method** (volunteer shares, chat dump, gallery walk, breakout report-back)

## Exercise selection by group size

| Size | Best formats |
|---|---|
| <15 | Pair share, hot seats, round-robin |
| 15–50 | Breakouts, gallery walks, polls, small-group report-back |
| 50+ | Chat, polls, solo reflection with volunteer shares |

## Facilitator guide format (SAY / DO / SHOW / TRANSITION)

For **every block**, produce:

```
BLOCK: [name]  ([start]–[end], [duration])
SAY:        [opening line verbatim, 1–3 sentences]
DO:         [what the facilitator physically does — start timer, open breakout, share screen]
SHOW:       [slide number / worksheet page / screen element]
WATCH-FOR:  [signals attendees are stuck / engaged / confused]
IF-BEHIND:  [what to cut or compress]
TRANSITION: [bridging line to next block]
```

## "CUT IF BEHIND" — minimum 2 markers

Mark at least 2 cuttable sections explicitly. Candidates:
- Second example in a teaching block
- Optional bonus exercise variant
- One hot-seat round
- Extended debrief on Exercise 1

## Attendee worksheet outline

Specify **page by page**, what each page contains. Typical structure:
1. Cover + promise + space for "my State B"
2. Framework reference (one diagram / checklist)
3. Exercise 1 working pages
4. Exercise 2 working pages
5. Next steps + commitment statement

## Workshop-specific production additions

- Materials list checked 48 h before (printed worksheets, digital PDF, tools)
- Breakout room plan (group size, assignment method, timer)
- Worksheet distribution tested (download link + DM backup)
- Timer/poll tool set up on platform
- Facilitator cheat-sheet: exercise prompts + timings on one page

Hand AV cues and contingencies to `event-run-of-show`.
