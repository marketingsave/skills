# Tech Checklist — Workshop Mode (hands-on, virtual or in-person)

Load this when `mode = workshop`. Workshops add **exercise delivery** and **interaction infrastructure** on top of the basic live-session tech.

## Worksheet / Materials Delivery

```
[ ] Attendee worksheet finalized (PDF, fillable if possible)
[ ] Worksheet emailed 24 h before with clear "print or open digitally" note
[ ] Backup link to worksheet posted in chat at workshop start
[ ] Any required tools listed in pre-work email (e.g., "bring your ESP login")
[ ] Pre-work confirmation — ask attendees to reply with a yes before the session
```

## Virtual Workshop (Zoom / StreamYard / similar)

```
[ ] All basic audio / camera / screen share checks from tech-webinar.md
[ ] Breakout rooms pre-configured with size + duration + prompt loaded
[ ] Breakout auto-assignment tested (pairs vs. small groups)
[ ] Timer tool selected (on-screen visual countdown for exercises)
[ ] Poll questions pre-built (one per exercise if used for debrief)
[ ] Chat prompts scripted in facilitator guide
[ ] Co-host assigned to monitor chat during exercises
[ ] Whiteboard / annotation enabled (if part of an exercise)
[ ] Screen share permission enabled for attendees if they'll share work
```

## In-Person Workshop

```
[ ] Room layout supports exercise format (round tables / pods / open floor)
[ ] Printed worksheets at every seat
[ ] Markers, sticky notes, flip charts staged (if used)
[ ] Visible timer (physical or projected) for exercise countdowns
[ ] Gallery-walk wall space cleared (if used for debrief)
[ ] Pairs / groups pre-assigned or a fast method to pair chosen
```

## Exercise Infrastructure

```
[ ] Every exercise has its EXACT prompt scripted in the facilitator guide
[ ] Every exercise has a stated duration (activity + debrief separately)
[ ] Every exercise has a debrief method (volunteer share / popcorn / gallery walk / poll)
[ ] Transition from exercise → debrief is scripted (not improvised)
[ ] At least 2 "CUT IF BEHIND" markers placed on agenda
```

## Interaction Cadence

```
[ ] No lecture block longer than 15 min without interaction (enforce in agenda)
[ ] Virtual: 5-min break every 45 min scheduled
[ ] Virtual: interaction every 10 min maximum (chat / poll / reaction / breakout)
[ ] Engagement prompts written into facilitator guide, not left to the moment
```

## Backup / Recovery

```
[ ] If breakouts fail: fallback solo-reflection version of each exercise ready
[ ] If worksheet fails to display: alternate delivery (read prompts aloud, chat)
[ ] If facilitator's internet drops: co-host briefed to continue from exercise prompts
[ ] If running 10+ min behind: pre-marked cut sections ready
```

## Post-Workshop

```
[ ] Follow-up email includes filled-worksheet request (engagement hook)
[ ] Recording + worksheet link delivered within 24 h
[ ] Soft upsell or next-step action matches the transformation delivered
```
