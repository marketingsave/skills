# Case Study — Templates

Copy-paste templates for each section. Replace bracketed tokens with real values. Every template assumes Phase 1 data is collected and confirmed.

---

## Headline Template

Formula: **"[Result] for [Client Type] in [Timeframe]"**

```
[NUMBER + UNIT] [OUTCOME NOUN] for [CLIENT DESCRIPTOR] in [TIMEFRAME]
```

Variants:
- `[%] [Metric] Increase for a [Industry] [Business Type] in [N Months]`
- `$[Amount] in [Outcome] for a [Client Type] in [N Days]`
- `[N]x [Metric] for a [Stage] [Client Type] in [Timeframe]`

Filled examples:
- `215% Social Engagement Increase for a 12-Location Hotel Chain in 3 Months`
- `$47K in New Pipeline for a B2B Consulting Firm in 90 Days`
- `3x Conversion Rate for a DTC Skincare Brand in 6 Weeks`

Rejection rule: if the headline has no number, rewrite. Blog-post-style headlines ("How We Helped X Succeed") are always rejected.

## Snapshot Box Template

Place immediately after the headline, before any prose.

```markdown
| Metric | Result |
|--------|--------|
| [Primary metric] | [Before] to [After] ([% delta]) |
| [Secondary metric] | [Absolute number + unit + timeframe] |
| [Efficiency metric] | [Before] to [After] |
| [Bonus metric] | [Number + context] |
```

Rules: 3-4 rows, lead row = headline metric, each row must contain a number, keep each cell under 10 words.

## Situation → Challenge → Action → Result (SCAR) Structure

Long-form body skeleton. Use as a drop-in template after the snapshot.

```markdown
## The Challenge

[Client Company], a [industry + size descriptor], was [situation in 2-3 sentences establishing baseline and stakes]. [Main contact name and role] had [specific problem in client's language].

They had already tried [previous attempt 1] and [previous attempt 2], but [why it failed]. The cost of inaction was [quantified business impact: $X lost, Y hours wasted, Z opportunities missed].

## The Solution

We delivered [one-sentence high-level approach] through a [N]-phase engagement:

**Phase 1: [Name] (Week/Month [range])**
[What was done + specific deliverable + key decision]

**Phase 2: [Name] (Week/Month [range])**
[What was done + specific deliverable + key decision]

**Phase 3: [Name] (Week/Month [range])**
[What was done + specific deliverable + key decision]

[Optional: Phase 4]

The key decision was [why this approach over alternatives]. [Client] specifically needed [constraint], so we [adaptation].

## The Results

**[Headline Metric]: [Before → After, % delta]**
[1-2 sentences of context: what changed, how it was measured.]

**[Metric 2]: [Absolute number + unit]**
[Context sentence.]

**[Metric 3]: [Before → After]**
[Context sentence.]

**[Unexpected Win — optional]**
[1-2 sentences on a secondary outcome the client did not request.]

> "[Direct client quote, 1-3 sentences]."
>
> — **[Name]**, [Role], [Company]

---

**Ready to [outcome mirroring headline result] for your [client type]?** [CTA link]
```

## Short-Form Template (300-500 words)

Compress SCAR into tighter blocks. No phase breakdown — collapse solution into one paragraph.

```markdown
# [Headline with number]

| Metric | Result |
|--------|--------|
| [M1] | [R1] |
| [M2] | [R2] |
| [M3] | [R3] |

**Challenge.** [Client + situation + problem + failed attempt, 3-4 sentences total.]

**Solution.** [What we delivered, 3-4 sentences, naming specific deliverables.]

**Results.** [Metric 1 with before/after.] [Metric 2 with before/after.] [Metric 3 with before/after.]

> "[Client quote]." — **[Name]**, [Role]

**[CTA question]?** [Link]
```

## One-Pager Template (200-300 words)

For PDF leave-behinds. Designed for single-page visual layout.

```markdown
# [Headline]
### [Client Name] — [Industry]

**The Numbers**
- [Metric 1]: [Result]
- [Metric 2]: [Result]
- [Metric 3]: [Result]

**The Problem**
[2 sentences — situation + pain point with number.]

**What We Did**
[3 sentences — named deliverables with timeframe.]

**The Outcome**
[2 sentences — headline result + one secondary win.]

> "[Short pull quote]." — [Name], [Role]

[CTA line + link]
```

## Testimonial Quote Block

```markdown
> "[Quote, 1-3 sentences, verbatim where possible]."
>
> — **[Full Name]**, [Role], [Company]
```

Paraphrased variant (when no verbatim quote exists):

```markdown
According to [Name], [Role] at [Company], the [specific outcome] was [paraphrased impact].
```

Never fabricate. If no quote and no paraphrase source, omit the block entirely.

## Call to Action Templates

Reference the specific result type from the case study.

- Service outcome: `**Ready to [headline outcome] for your [client type]?** [Link]`
- Discovery call: `**Want to see what's possible for your [business]?** [Book a call](link)`
- Lead magnet: `**Get the [framework name] we used with [Client]:** [Download](link)`

Use `[your-link-here]` when the user has not supplied a destination.
