# Production Cues, Shot List, Pre-Filming Checklist

Load this reference when writing Phase 3 (full script with cues) or Phase 4 (shot list + pre-filming checklist).

## Production Cue Format

| Cue | When to Use | Example |
|-----|-------------|---------|
| `[CUT TO: shot type]` | Camera angle / framing change | `[CUT TO: tight shot, direct eye contact]` |
| `[B-ROLL: description]` | Supplementary footage over narration | `[B-ROLL: screen recording of Notion kanban with 3 tasks]` |
| `[TEXT ON SCREEN: text]` | Key phrases, stats, overlays | `[TEXT ON SCREEN: "< 2 minutes? Do it NOW."]` |
| `[SFX: description]` | Sound effect or music change | `[SFX: upbeat stinger]` |
| `[PAUSE: Xs]` | Intentional beat for emphasis | `[PAUSE: 1s]` |
| `[TRANSITION: type]` | Cut, zoom, swipe, whip pan | `[TRANSITION: whip pan left]` |

**Specificity rule:** every B-ROLL cue must name the exact asset. "[B-ROLL: something relevant]" is useless.

## Shot List Template

Append a table listing every camera setup and visual asset:

```
## Shot List

| # | Shot Type | Description | Duration | Notes |
|---|-----------|-------------|----------|-------|
| 1 | Tight shot | Direct eye contact, hook line | 3s | High energy |
| 2 | Medium shot | Talking head, setup + body | 20s | Main delivery |
| 3 | Screen recording | [specific asset] | 5s | B-roll over body |
| 4 | Tight shot | CTA, point at camera | 2s | Same framing as hook |

**Total setups:** [X camera positions + Y screen recordings]
**Estimated filming time:** 10-20 minutes (with retakes)
```

## Pre-Filming Checklist

```
## Pre-Filming Checklist

- [ ] Script printed or on teleprompter/phone
- [ ] Camera framing set for each shot type
- [ ] Audio levels tested (lapel or phone mic positioned)
- [ ] Lighting consistent across talking head shots
- [ ] B-roll / screen recordings captured and labeled
- [ ] Trending audio saved/downloaded (if using)
- [ ] CTA link/resource is live and tested before publishing
- [ ] Caption and hashtags pre-written in notes app
```
