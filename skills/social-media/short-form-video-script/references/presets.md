# Platform Presets — Detailed

Load this when the deliverable is platform-specific (tiktok, reels, or shorts) or when generating multi-platform repurposing notes in Phase 4.

## Preset Table

| Preset | Platform | Length Sweet Spot | Hook Window | Visual Change | Audio Convention | CTA Style |
|--------|----------|-------------------|-------------|---------------|------------------|-----------|
| **`tiktok`** | TikTok | 15-60s | First 1-2s | Every 2-3s | Trending sound REQUIRED (or trending "original sound" format) | "Follow for more" + on-screen text; stitch/duet prompts welcome |
| **`reels`** | Instagram Reels | 15-60s | First 1-2s | Every 2-3s | Trending audio preferred; licensed music OK | Follow + save + share; cover image matters |
| **`shorts`** | YouTube Shorts | 30-60s | First 3s (slightly longer hook tolerance) | Every 3-4s | Original voiceover or royalty-free music; trending audio optional | Subscribe + suggested video; end screen prompt |
| **`multi`** | All three | 30s or 60s | First 1-2s | Every 2-3s | Trending audio with royalty-free fallback | Platform-specific CTA variants in deliverable |

**Preset effect on the script:** the preset changes hook window, audio suggestion, CTA phrasing, and the repurposing notes in Phase 4 — the overall Brief → Structure → Write → Polish flow is identical.

---

## TikTok Preset

- **Audio:** Trending sound or trending "original sound" format — non-negotiable for reach. Check the TikTok Creative Center weekly.
- **Captions:** Short, punchy, often a question. Emojis OK. First line shows in-feed.
- **Hashtags:** 3-5 mixed (broad + niche + trend-specific). Avoid banned/shadow-tagged ones.
- **CTA:** "Follow for more," stitch/duet prompts ("Duet this with your version"), comment hooks ("Drop a 1 if you want part 2").
- **Native quirks:** Green screen format, text-to-speech voiceover, #POV frames all boost reach.

## Instagram Reels Preset

- **Audio:** Trending audio from the Reels library (flagged with an up-arrow). Licensed music OK if trending fails.
- **Cover image:** Design a deliberate cover (readable thumbnail text) — Reels grid is a portfolio.
- **Captions:** Longer captions perform well. Hook in line 1, value in body, CTA at bottom.
- **CTA:** Save + share are weighted more than follow. "Save this for later" / "Send to a friend who needs this."
- **Cross-promotion:** Post Story teaser, pin to profile grid if evergreen.

## YouTube Shorts Preset

- **Audio:** Original voiceover preferred for watch time. Royalty-free background music optional. Trending audio less critical than TikTok/Reels.
- **Hook tolerance:** Slightly longer hook window (up to 3s) — YouTube viewers scroll less aggressively.
- **CTA:** Subscribe + suggested-video link. Pin a comment with a link to the long-form companion video.
- **End screen:** Last 2 seconds — point to another Short or a long-form video.
- **SEO:** Title + description matter more than on TikTok/Reels. Include searchable keywords.

## Multi Preset — Repurposing Notes Template

When delivering `multi`, include a block like this:

```
### Repurposing Notes

**TikTok:** [Trending sound name] + #tag1 #tag2 #tag3. Try stitching [related creator format].
**Instagram Reels:** Use cover image with text "[short phrase]". Pin to grid. Cross-post to Stories with poll sticker.
**YouTube Shorts:** Swap CTA to "Subscribe for more [niche] tips." Pin comment linking to long-form video on [topic].
```
