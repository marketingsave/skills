# Pricing Frameworks (Proposal Mode — Primary)

In `proposal` mode, choose the framework that matches the engagement. Default to **fixed-price** unless specified otherwise. In `sow` mode, pricing is already decided — just document it with a clear schedule.

## Fixed-Price (Default)

Best for: clear scope, defined deliverables, known end date.

```
## Investment

**Project Investment: $X,XXX**

This is a fixed-price engagement covering all deliverables listed above:

- [Deliverable 1]
- [Deliverable 2]
- [Deliverable 3]
- [X] rounds of revisions
- [X] weeks of post-launch support

**Payment Schedule:**
- 50% ($X,XXX) due upon proposal acceptance
- 50% ($X,XXX) due upon final delivery

Additional revisions beyond [X] rounds: $XXX per round.
```

## Retainer (Monthly)

Best for: ongoing services, marketing, support, advisory with no fixed end date.

```
## Investment

**Monthly Retainer: $X,XXX/month**

This retainer covers:

- [Included service 1] (up to [quantity] per month)
- [Included service 2] (up to [quantity] per month)
- [Included service 3]
- [Response time commitment]

**Minimum commitment:** [X] months
**Billing:** Monthly, invoiced on the 1st, due within 15 days

Additional work beyond the retainer scope is billed at $XXX/hour with prior approval.

Either party may cancel with 30 days written notice after the minimum commitment period.
```

## Tiered (Good / Better / Best)

Best for: unknown budget, flexibility. **Always recommend the middle tier.**

```
## Investment

We have structured three options based on scope and investment:

### Option 1: [Tier Name] — $X,XXX
- [Deliverable 1]
- [Deliverable 2]
- [X] revision rounds
- Best for: [who this tier suits]

### Option 2: [Tier Name] — $X,XXX (Recommended)
- Everything in Option 1, plus:
- [Additional deliverable]
- [X] revision rounds
- Best for: [who this tier suits]

### Option 3: [Tier Name] — $X,XXX
- Everything in Option 2, plus:
- [Premium deliverable]
- [Ongoing support period]
- Best for: [who this tier suits]

**Payment Schedule:**
- 50% due upon proposal acceptance
- 50% due upon final delivery
```

## Pricing Rules

- NEVER list hourly rates in a fixed-price proposal. Client is buying an outcome.
- Round to clean numbers. $5,000 not $4,850.
- Middle tier 1.5x-2x base. Top tier 2x-3x base.
- Descriptive tier names: "Foundation / Growth / Scale" — NOT "Bronze / Silver / Gold."
