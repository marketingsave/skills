# Formato: WhatsApp — API Oficial (Privado)

## Perguntas de briefing
- Produto promovido?
- Objetivo? (reativação, lançamento, urgência)
- 3 etapas do fluxo?
- Dados/provas reais?
- CTA final?

## Padrões de entrega
Fluxo de **3 mensagens**:
1. **Abertura** — tom de AVISO / INFORMAÇÃO / ATUALIZAÇÃO. Gera curiosidade para clicar em "SABER MAIS". Nunca parece venda na largada.
2. **Conteúdo/marketing** — copy principal com tom de venda, já com permissão ganha na mensagem 1.
3. **CTA/urgência** — fechamento com ação clara, prazo, razão para agir agora.

### Regras
- Máximo de **500 caracteres** por mensagem (mais restritivo que grupo).
- Tom mais pessoal — é 1:1, privado, não broadcast.
- Formato de código para colar no WhatsApp (`*negrito*`, `_itálico_`).

## Super Prompt de briefing

> Preciso de um fluxo de 3 mensagens para WhatsApp privado via API oficial. Antes de escrever, me informe: (1) Produto? (2) Objetivo do fluxo? (3) Dados e provas reais disponíveis? (4) CTA final? A primeira mensagem deve ter tom de AVISO/ATUALIZAÇÃO para gerar curiosidade para o lead clicar em "SABER MAIS". Máximo de 500 caracteres por mensagem. Entregue em formato de código para copiar e colar.

## Checklist específico
- [ ] 500 caracteres respeitados em cada uma das 3 mensagens.
- [ ] Mensagem 1 não tem cara de venda — parece aviso/update.
- [ ] Tom pessoal (privado), não de grupo.
- [ ] CTA final com razão para agir agora.
