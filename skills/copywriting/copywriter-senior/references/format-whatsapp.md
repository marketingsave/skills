# Formato: WhatsApp — Grupos

## Perguntas de briefing
- Objetivo do fluxo? (aquecimento, venda, urgência, pós-compra)
- Produto/serviço promovido?
- Público do grupo?
- Tom da comunicação do expert?
- Provas reais (números, dados, depoimentos)?
- Quantas mensagens?

## Padrões de entrega
- Mensagens em **formato de código** prontas para colar no WhatsApp: `*negrito*`, `_itálico_`, `~riscado~`.
- Máximo de **900 caracteres** por mensagem.
- Conexão fluida entre as mensagens (uma puxa a próxima).
- Sempre entregar em artefato.

## Super Prompt de briefing

> Preciso escrever um fluxo de mensagens para WhatsApp de grupos. Antes de começar, me informe: (1) Qual é o produto/serviço? (2) Qual é o objetivo do fluxo? (3) Quantas mensagens? (4) Qual é o público do grupo? (5) Tem dados ou provas reais para usar? (6) Qual é o tom do expert? Com essas informações, escreva as mensagens em formato de código prontas para copiar e colar no WhatsApp, com no máximo 900 caracteres cada, com conexão fluida entre elas. Entregue em artefato.

## Checklist específico
- [ ] Limite de 900 caracteres respeitado em cada mensagem.
- [ ] Formatação WhatsApp correta (`*`, `_`, `~`).
- [ ] Transições entre mensagens (curiosidade para próxima).
- [ ] Entregue em bloco de código, pronto para colar.
