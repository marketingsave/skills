# Formato: Briefing de Campanha

Briefing é o documento que guia toda a campanha. Um copywriter sênior **escreve briefing como se fosse copy** — com linguagem viva, ângulo definido, público claro.

## Quando usar
- Início de lançamento.
- Reorganização de campanha em andamento.
- Passagem de contexto para outro redator/agência.

## Entradas necessárias
- Todos os documentos de pesquisa disponíveis (avatar, dores, desejos).
- Dados do produto: oferta, preço, garantia, bônus.
- Histórico: o que já foi feito, o que funcionou, o que não.
- Referências do expert: falas, aulas, entrevistas.
- Concorrentes e diferenciação.

## Estrutura típica do briefing
1. **Produto e oferta** — o que é, por quanto, o que inclui.
2. **Avatar** — quem é, o que sente, o que já tentou.
3. **Nível de consciência** (Schwartz).
4. **Big idea / ângulo central** — o conceito que amarra a campanha.
5. **Promessa principal** — o que o cliente leva.
6. **Objeções mapeadas** e como serão quebradas.
7. **Provas disponíveis** — depoimentos, números, autoridade.
8. **Tom e voz** — como o expert fala.
9. **Peças a serem produzidas** — lista de entregáveis.
10. **Cronograma** — o que entra no ar quando.

## Super Prompt de briefing

> Preciso de um briefing completo de campanha. Envie todos os documentos de pesquisa, dados e referências disponíveis. Com base nesses materiais, crie um briefing com linguagem única para este projeto, escrito por um copywriter e pesquisador de verdade, seguindo a estrutura do documento de referência enviado.

## Checklist específico
- [ ] Big idea formulada em uma frase.
- [ ] Avatar com nome, idade, situação concreta.
- [ ] Objeções reais (não genéricas).
- [ ] Linguagem do próprio expert usada no documento.
- [ ] Lista de entregáveis com prazos.
