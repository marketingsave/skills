# Hybrid Mode — Deep Reference

Group coaching, fitness, wellness, or mixed-format. Blends live teaching, community, and peer accountability without cohort rigidity.

## Brief — Mode-Specific Inputs

- Balance of live teaching vs. community-driven time
- Hot seat rotation plan

## Hybrid Brief Template

```
## Hybrid Group Program Brief
**Program:** [Name]
**Outcome:** [What members achieve]
**Duration:** 8 weeks
**Group size:** 8-15
**Format:** Weekly 60-75 min live group call + community + async assignments
**Price:** $749-$949/person
```

## Framework — Loose Weekly Themes + Group Call Rhythm

```
## Curriculum Outline
### Week 1: [Foundation Topic]
- Teaching: [Core concept]
- Assignment: [What participants do]
- Community prompt: [Discussion question]

### Week 2: [Building Block Topic]
[...]

### Final Week: [Integration / Next Steps]
- Teaching: [Putting it all together]
- Assignment: [Personal action plan]
- Celebration: [Group wins and commitments]
```

## Weekly Group Call Agenda (60-75 min)

```
Opening (5 min):     Check-in round — one word describing your week
Wins/Progress (10):  Members share wins since last call
Teaching (15-20):    This week's core content or framework
Hot Seat (20-25):    2-3 members get focused coaching
Q&A (10):            Open floor
Closing (5):         This week's assignment and commitment share
```

## Engagement Tactics (hybrid default set)

1. Welcome thread — intros + #1 goal
2. Weekly wins channel
3. Accountability pairs — weekly check-ins
4. Hot seat rotation — 2-3 per call
5. Mid-week challenge prompts
6. One guest expert session

## Operations

**Recruitment:**
- Rolling or cohort-based enrollment
- Application or enrollment form
- Always maintain a waitlist for next cohort

**Onboarding (mode-specific):** accountability pair assignments, curriculum release schedule.

**Pricing Framework:**
```
Per-person pricing:
- Desired hourly rate × total program hours = base cost
- Base cost ÷ minimum group size = per-person floor price
- Add value premium for community, accountability, curriculum

Example (8-week hybrid):
- Total delivery time: 24 hours over 8 weeks
- Hourly rate: $150 → base cost: $3,600
- Minimum group of 8: $450/person floor
- Value premium: $300-500/person
- Program price: $749-949
- Revenue at 12 participants: $8,988-$11,388
```

Payment options: Full-pay (10% discount), 2-pay split (50% now / 50% week 4), monthly installments (slight premium).

## Assessment — Completion + Continuity

- Celebration/graduation event in final week
- Mid-point and end-of-program feedback surveys
- Next-cohort enrollment offer with alumni discount
- Testimonial collection

## Hybrid Health Targets

| Metric | Target |
|--------|--------|
| Attendance | 70%+ group calls |
| Completion | 75%+ assignments |
| Satisfaction (NPS) | 40+ |
| Repeat / next-cohort | 25%+ |

## Anti-Patterns (Hybrid)

- **All content, no interaction** — if members never talk to each other, it is a course, not a group program.
- **Underpricing** — do not price like a $27 course.

## Recovery (Hybrid)

- **Under minimum enrollment**: delay start, extend marketing, or run smaller with adjusted pricing.
- **Drop-off after week 2-3**: increase touchpoints, reach out personally, review content relevance.
- **Content not landing**: mid-program survey, adjust remaining weeks.
- **No community engagement**: post conversation starters, share your progress, create low-effort prompts (polls, quick wins).
